# Repository Guide

This repository is a Vite/React music player project. Keep changes scoped to the requested feature or fix, and prefer the existing npm scripts.

## Project Commands

- `npm run dev` - start the Vite dev server
- `npm run build` - run TypeScript build and Vite production build
- `npm run lint` - run ESLint
- `npm test` - run Vitest once
- `npm run test:watch` - run Vitest in watch mode
- `npm run test:coverage` - run Vitest with coverage

Before committing, run the narrowest relevant checks. For broad frontend changes, run `npm run build` and `npm test` when practical.
